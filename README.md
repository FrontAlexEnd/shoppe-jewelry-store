> ### Посмотреть сайт: https://frontalexend.github.io/shoppe-jewelry-store

> 🔘 **Избранные товары, которые хранятся на бекенде (strapi)** <br>
> 🔘 **Страница каждого товара с уникальным описанием и реальными отзывами** <br>
